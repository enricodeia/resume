Welcome to Resume Builder!

This interactive tool allows you to create a visually engaging, animated resume. Here's how to get started:

  • Add Sections: Choose from templates in the sidebar
  • Arrange: Drag sections to position them
  • Customize: Select a section to edit its content and animation
  • Preview: See how your animated resume will appear
  • Export: Download the HTML file to use anywhere
